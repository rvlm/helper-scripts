RVLM Helper Routines
====================

This project is another set of helpers, a home-grown under-framework, created
to make development in Python not that aesthetically painful as it has always
been for my after I've finally discovered that a painful abyss of desperation
this language is. This code is supposed to be used for various experiments
during my PhD research in radio physics at Voronezh State University, Russia.
